# Brittany Herbert – Developer Portfolio

Live Site: https://brittany-herbert-portfolio.vercel.app/

A modern, responsive developer portfolio built to showcase full-stack projects, technical skills, and real-world problem solving.

This site highlights applications I’ve designed and built across the stack, with a focus on clean UI, scalable architecture, and practical functionality. It also reflects my growing focus in cybersecurity and secure development practices.

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion

## Features
- Category-based project filtering
- Responsive design across devices
- Animated UI with Framer Motion
- Clean, modular component structure

## Featured Projects
- **Vibexperience** – Event coordination platform (Node.js, MongoDB)
- **TruGrind** – Fitness tracking app with authentication (MERN stack)
- **CoinWatch** – Cryptocurrency tracker (Django, PostgreSQL, CoinGecko API)

## Deployment
Deployed on Vercel with automatic CI/CD via GitHub.
